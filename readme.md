# Time Cache
>本项目基于rust,实现了一个简单而高效时序内存数据库，实现原理即将数据存储在一个大的Hashmap中，通过键值队的方式快速进行索引数据

## 实现功能:
| 方法       | 是否完成     | 描述     |
|----------|----------|--------|
| Create   | &#10003; | 创建一个队列 |
| Set      | &#10003; | 插入一个值  |
| Get      | &#10003; | 查找最新值  |
| Range    | &#10003; | 查找范围值  |
| Query    | &#10003; | 查询指定时间值 |
| SetArray | &#10005; | 插入一组值  |
| SetMuti  |     &#10005;     | 插入多值   |
